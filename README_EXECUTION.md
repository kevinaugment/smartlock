## 🚨 重要提醒

### 执行原则

1. **一次一页**：不要同时处理多个页面
2. **完成再继续**：每页验收后再进入下一页
3. **Git常提交**：每页提交一次，不要积累
4. **记录进度**：每页完成立即更新日志
5. **保持节奏**：累了就休息，保证质量

### 常见错误避免

❌ **不要做**：
- 批量删除多个文件后一起提交
- 跳过备份直接删除
- 忘记配置重定向
- 不更新进度日志
- 提交message含糊不清

✅ **应该做**：
- 每页独立处理和提交
- 严格按照7步流程
- 每步验证再继续
- 详细记录进度
- 清晰的Git历史
