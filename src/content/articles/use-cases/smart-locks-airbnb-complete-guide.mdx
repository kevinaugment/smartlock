## Operational Best Practices

### Code Management Strategy

**Single code vs unique codes:**

| Strategy | Pros | Cons | Best For |
|----------|------|------|----------|
| **Unique per guest** | ‚úÖ Audit trail<br>‚úÖ Security<br>‚úÖ Professional | ‚ùå More management<br>‚ùå Potential PMS integration needed | Most hosts |
| **Rotating weekly** | ‚úÖ Simple<br>‚úÖ Less management | ‚ùå Less security<br>‚ùå No audit trail | Very small scale |
| **Same code always** | ‚úÖ Ultra simple | ‚ùå Very insecure<br>‚ùå No accountability | ‚ö†Ô∏è Not recommended |

**Recommended:** Unique code per reservation

**Code format:**
- 4-6 digits (balance security and memorability)
- Avoid: 1234, 0000, repeating patterns
- Avoid: Dates, addresses
- Use lock's random generation

### Battery Management

**Prevent battery lockouts:**

‚úÖ **Set calendar reminders**
- WiFi locks: Every 3 months
- Zigbee/Z-Wave: Every 12 months

‚úÖ **Monitor battery level**
- Check app weekly
- Replace at 30% (not 10%)
- Keep spare batteries on-site

‚úÖ **Guest arrival timing**
- Never replace batteries on turnover day
- Do it 1-2 days before next guest

‚úÖ **Emergency backup**
- Physical key hidden securely
- 9V battery emergency power (if supported)
- Local contact with spare key

### Maintenance Schedule

**Monthly:**
- Check battery level
- Test lock operation (lock/unlock)
- Clean keypad (fingerprints, dirt)
- Verify codes are expiring correctly

**Quarterly:**
- Replace WiFi lock batteries
- Lubricate lock mechanism
- Tighten installation screws
- Test physical key backup

**Annually:**
- Replace Zigbee/Z-Wave batteries
- Deep clean lock exterior
- Check door alignment
- Update firmware if available

## Common Guest Issues & Solutions

### Issue 1: Guest Can't Get Code to Work

**Troubleshooting script for guests:**

"Try these steps:
1. ‚úÖ Enter code slowly (1 second between digits)
2. ‚úÖ Press # or ‚úì after code
3. ‚úÖ Turn handle immediately after beep
4. ‚úÖ Verify you're at correct door
5. ‚úÖ Check code start time (4pm check-in)"

**Remote diagnostics:**
1. Check if code is active in app
2. Verify dates/times correct
3. Check battery level
4. Send test unlock command
5. Regenerate code if needed

**Emergency backup:**
- Physical key location
- Neighbor with backup key
- Your arrival time if needed

### Issue 2: Door Won't Lock

**Common causes:**
- ‚ö†Ô∏è Door not fully closed
- ‚ö†Ô∏è Strike plate misaligned
- ‚ö†Ô∏è Deadbolt binding
- ‚ö†Ô∏è Low battery

**Guest instructions:**
"Push door firmly closed until you hear lock engage. If still not locking, try these:
1. Close door harder (may need firm push)
2. Lift door handle while closing
3. Check nothing is blocking door frame
4. Call host if persists"

### Issue 3: Guest Locked Out

**Prevention:**
- Auto-lock enabled
- Codes don't expire until 11am checkout day
- Physical key backup
- Emergency contact clearly stated

**Response protocol:**
1. Verify guest identity
2. Generate temporary emergency code
3. Send via text/call immediately
4. Guide through entry
5. Follow up to prevent recurrence
